Toolset

Here are some of my favorite development tools. Although not listed, I really enjoy experimenting with all sorts of technologies. I believe there is no single tool to rule them all. It's really about selecting the right tool for the job at hand.

Python

Python is great all-purpose programming language. This is my go-to language for quickly getting projects up and running. I love the community support and the amount of useful packages. It is especially great for data processing and machine learning tasks.

C

C is an excellent low-level programming language. I especially like working with it when memory management is a concern. It is also extremely fast, portable, and works well with embedded systems. I like using it when developing cryptographic systems due to the natural use of pointers and bit manipulation.

Java

I use Java from time to time simply due to its prevalence. It is one of the most popular programming languages and drives many systems. I appreciate its objected-oriented approach and ease of use. If I am not using Python or C, I definitely lean towards Java.

Docker

Docker has quickly become my favorite tool since I started using it. When developing any major project, I always containerize it with Docker. My favorite things about Docker include its portability and resource management. In addition, it is well suited for a collaborative environment. Any new developer can get a project up and running locally in no time.

Linux

I always prefer to use a Linux-based OS. The overall architecture is great and its command line interface makes development a joy. My preferred setup is WSL with Ubuntu. I especially prefer it when operating in a headless enviroment.

R

R is a programming language that is great for working with data. You can easily preprocess, clean, and prepare data in just a few lines of code. In addition, you can perform and statistical analysis and visualization. You can also build pretty robust machine learning models with it, although I typically go to Python for modeling.
